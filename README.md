Add to your Cursor `mcpServers.json` configuration:
```json
{
  "mcpServers": {
    "mcp-repl": {
      "command": "npx",
      "args": [
        "-y", "mcp-repl"
      ],
      "env": {},
      "disabled": false,
      "autoApprove": [
        "executenodejs",
```


### Meta-variables

- `$$$ARGS` - Multiple arguments
- `$$$` - Any content
